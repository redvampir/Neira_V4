# ADR Index

Этот каталог хранит **Architectural Decision Records (ADR)** — формальные записи архитектурных решений.

## Правила
- Нумерация: `0001-...`, `0002-...` и т.д.
- Статусы: `Proposed`, `Accepted`, `Superseded`, `Deprecated`.
- Если ADR заменён, в колонке «Заменён чем» указывается номер нового ADR.
