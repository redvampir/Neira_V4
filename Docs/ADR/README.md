# Architectural Decision Records (ADR)

Реестр архитектурных решений проекта Neira.

## Правила статусов

- `proposed` — предложено, ждёт подтверждения.
- `accepted` — принято и обязательно к исполнению.
- `superseded` — заменено более новым ADR.

## Как добавлять новый ADR

1. Создать файл `NNNN-short-title.md` из `template.md`.
2. Добавить строку в индекс.
3. Если решение заменяет старое, пометить старое как `superseded` и указать ссылку на новое.
